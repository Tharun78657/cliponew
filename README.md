# Clipo Media - Premium Digital Experience

A cutting-edge, visually immersive website for **Clipo Media**, a creative production studio based in Hyderabad. This project showcases a "liquid glass" aesthetic, featuring advanced glassmorphism effects, fluid animations, and a high-performance interactive user interface.

## üåü Key Features

*   **Liquid Glass Aesthetic**: A premium, modern design language utilizing backdrop filters, subtle gradients, and noise textures to create a realistic glass effect.
*   **Dynamic Theme Switcher**: 
    *   **Light Mode**: Clean and airy.
    *   **Dark Mode**: Sleek and professional.
    *   **Dim Mode**: A unique, deep midnight blue theme.
    *   *Includes a custom "jelly" animation and expanding ripple effect when switching themes.*
*   **Interactive Navigation**:
    *   "iPhone-style" dynamic dock navigation.
    *   Smooth active state transitions with a "liquid bubble" indicator.
    *   IntersectionObserver-based scroll spying for high-performance active section highlighting.
    *   Draggable navigation bubble for enhanced interactivity.
*   **Responsive Design**: Fully optimized for all devices, from large desktops to mobile screens.
*   **Smooth Animations**: Custom CSS animations and transitions for a fluid user experience without performance lag.

## üöÄ Getting Started

### Prerequisites

You need a modern web browser to view this project as it uses modern CSS features like `color-mix` and `backdrop-filter`.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Gudapavansai/clipo-new.git
    cd clipo-new
    ```

2.  **Open the project**:
    Simply open the `index.html` file in your browser to view the site locally.
    
    *Or using a local server (recommended):*
    ```bash
    npx serve .
    ```

## üåê Deployment

This project is deployed and hosted on **Vercel**.

*   **Live Demo**: [https://clipo-pqg9q4rpz-pawan-sais-projects.vercel.app](https://clipo-pqg9q4rpz-pawan-sais-projects.vercel.app)

2.  Run `vercel deploy` in the project directory.
